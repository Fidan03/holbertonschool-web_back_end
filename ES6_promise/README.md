# ES6 Promise

This project covers JavaScript Promises: how to create them, chain `then`/`catch`, use static `Promise` methods, and work with `async`/`await` and `throw`/`try`.

## Learning Objectives

- Promises (how, why, and what)
- How to use `then`, `resolve`, `catch`
- How to use every method of the `Promise` object
- `throw` / `try`
- The `await` operator
- How to use an `async` function

## Requirements

- All files interpreted/compiled on Ubuntu 20.04 LTS using Node 20.x.x and npm 9.x.x
- Code uses the `.js` extension
- Code is tested using Jest (`npm run test`)
- Code is linted using ESLint
- Run `npm run full-test` to verify tests and lint together
- All functions must be exported
