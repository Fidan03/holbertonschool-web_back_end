# ES6 Data Manipulation

This project covers advanced array/object manipulation techniques introduced in ES6, including `map`, `filter`, `reduce`, Typed Arrays, `Set`, `Map`, and `WeakMap`.

## Learning Objectives

- How to use `map`, `filter` and `reduce` on arrays
- Typed arrays
- The `Set`, `Map`, and `WeakMap` data structures

## Requirements

- All files interpreted/compiled on Ubuntu 20.04 LTS using Node 20.x.x and npm 9.x.x
- Code uses the `.js` extension
- Code is tested using Jest (`npm run test`)
- Code is linted using ESLint
- Run `npm run full-test` to verify tests and lint together
- All functions must be exported
